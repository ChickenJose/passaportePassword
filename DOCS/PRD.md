# PRD - Passaporte Password

## Propósito e Público-alvo
Projeto inicial de Next.js 14 focado na criação de uma interface moderna e segura, possivelmente relacionada a gestão de identidades ou credenciais ("Passaporte Password"). O público-alvo são utilizadores que procuram uma experiência visual semelhante à plataforma Futuro Digital da CML.

## Objetivos e Funcionalidades Planeadas
- Interface base responsiva e atrativa.
- Integração de um design system inspirado em https://futuro-digital-cml.vercel.app/.
- Arquitetura robusta utilizando Next.js App Router, com TypeScript para tipagem estática.
- Suporte a rotas de API (backend) caso necessário futuramente.

## Regras de Design
- Utilizar Vanilla CSS para máximo controlo.
- Cores vibrantes, tema claro/escuro, micro-interações, tipografia moderna (ex: Inter).
- Aparência premium.
