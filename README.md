# Passaporte de Competências Digitais - Verificador de Senhas

Este site foi desenvolvido por **Bruno Sousa** para a **Câmara Municipal de Lisboa (CML)**.

O objetivo principal deste projeto é apoiar os cursos de inclusão digital do **Passaporte de Competências Digitais**, auxiliando os utilizadores a compreender a importância da segurança digital e a criar senhas (passwords) mais seguras e robustas.

## 💻 Tech Stack (Tecnologias Utilizadas)

- **Next.js (14)**: Framework React usado para estruturação, rotas e renderização da aplicação.
- **React (18)**: Biblioteca JavaScript principal para a construção da interface interativa e componentes de UI.
- **TypeScript**: Superset de JavaScript que adiciona tipagem estática, garantindo um código mais robusto e seguro.
- **CSS Nativo (Vanilla CSS)**: Estilização responsiva, acessível e moderna, alinhada com a identidade visual da CML (sem dependência de frameworks externos).

## ✨ Funcionalidades Principais

- **Avaliador de Senha em Tempo Real**: Mede o esforço e o tempo estimado para quebrar uma senha, utilizando métricas atualizadas baseadas na Hive Systems.
- **Assistente Interativo (Wizard)**: Processo passo-a-passo que orienta o utilizador na construção lógica e segura de uma password, ajudando o público-alvo a interiorizar as boas práticas tecnológicas.
- **Acessibilidade e Usabilidade**: Foco numa navegação fluída, com feedback imediato e design limpo adaptado a diferentes níveis de literacia digital.

## 🚀 Como executar o projeto localmente

1. **Instalar Dependências**
   Execute o seguinte comando na raiz do projeto:
   ```bash
   npm install
   ```

2. **Iniciar o Ambiente de Desenvolvimento**
   Arranque o servidor local com o comando:
   ```bash
   npm run dev
   ```

3. **Aceder no Navegador**
   Abra o endereço [http://localhost:3000](http://localhost:3000) e explore a aplicação.
